# Subset-Sum-Complexity-Research
Empirical study of instance-dependent complexity in the Subset Sum Problem using C++ and Python. Includes research paper and statistical analysis.
# Phase Transition in Subset Sum Problem

This repository contains the source code and the final research paper for an empirical study on the **Subset Sum Problem (SSP)**.

## 🔬 Research Overview
The study analyzes 2,400 experimental trials to demonstrate how data entropy influences computational complexity. Key findings include:
- Identification of a **Phase Transition** near critical density ($d \approx 1.0$).
- Efficiency of **Suffix-Sum Pruning** on structured vs. random datasets.
